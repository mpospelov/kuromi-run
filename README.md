# Kuromi Run 🖤💜

Браузерная 3D-игра для девочки 7 лет: Куроми бежит по трассе из трёх полос,
собирает черепушки и конфеты, объезжает препятствия. Проиграть нельзя —
при столкновении конфеты рассыпаются, и их можно тут же подобрать.
Каждое третье столкновение — «книжная переменка» с книжкой для 1 класса.

Всё сделано кодом: персонаж и мир из примитивов Three.js, музыка и звуки
генерируются через WebAudio. Никаких внешних ассетов.

## Запуск

```bash
npm install
npm run dev          # локально: http://localhost:5173
npm run dev -- --host  # чтобы открыть с планшета по адресу компьютера в сети
```

Сборка для хостинга (GitHub Pages и т.п.): `npm run build` → папка `dist/`.

## Управление

- **Клавиатура:** ← → / A D — полосы, ↑ / пробел / W — прыжок, ↓ / S — подкат, Esc — пауза
- **Планшет:** свайпы влево/вправо/вверх/вниз

## Трассы

1. **Конфетная долина** — медленная, ~1 минута
2. **Ночной город Куроми** — средняя, ~1,5 минуты
3. **Звёздное небо** — быстрая, ~2 минуты

На финише 1–3 звезды за 50/75/90 % собранных сладостей; рекорды хранятся
в localStorage браузера.

## Структура

- `src/main.js` — игровой цикл, ввод, столкновения, экраны
- `src/tracks.js` — темы трасс, декор (InstancedMesh), паттерны предметов
- `src/kuromi.js` — героиня из примитивов, процедурные анимации
- `src/audio.js` — WebAudio: мелодии трасс и эффекты
- `scripts/verify.mjs` — сквозная проверка игры в headless-браузере (`node scripts/verify.mjs` при запущенном dev-сервере)

Дизайн: handoff из Claude Design («Куроми Раннер»), реализация — Vite + Three.js.
